# Calculating FAIRsoft indicators

### Interactive evaluation

The FAIRsoft Evaluator allows you to compute FAIRsoft indicators for individual software tools through a graphical interface.

!!! example "This approach is useful when you want to"

    - Evaluate a single software tool
    - Review the metadata used in the evaluation
    - Provide missing information during the assessment
    - Obtain a detailed FAIRness report
    - Generate metadata files for a software tool (JSONLD and CFF)

→ **[View the interactive evaluation tutorial](./fairsoft_evaluator.md)**


### Programmatic evaluation

FAIRsoft indicators can also be computed programmatically using the Software Observatory APIs.

!!! example "This approach is useful for"
    - Large-scale analyses of software ecosystems
    - Benchmarking studies
    - Automated evaluation pipelines"

→ **[View the programmatic evaluation section](./programmatic_evaluation/index.md)**
