# The FAIRsoft Evaluator

The FAIRsoft Evaluator guides you through the evaluation of an individual software tool. You can start from a GitHub repository, existing Software Observatory metadata, or a metadata file, and then review and enrich the metadata before computing the indicators. 

→ **[Go to Software Evaluator](https://openebench.bsc.es/observatory/Evaluation/)**
